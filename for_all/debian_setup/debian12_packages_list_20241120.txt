# i3 WM
i3 (4.22-2)
	metapackage (i3 window manager, screen locker, menu, statusbar)
	This metapackage installs the i3 window manager (i3-wm), the i3lock screen locker,
	i3status (for system information) and suckless-tools (for dmenu).
	These are all the tools you need to use the i3 window manager efficiently.

	recomends:
		dunst, i3lock, suckless-tools
