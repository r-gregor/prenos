Convert current path to filename to "git" path

fname ... filename somewhere inside "~/majstaf/...", like:
/home/gregor.redelonghi/majstaf/majbin/git-commands/gupdate-file-en
/home/gregor.redelonghi/majstaf/majbin/zze
/home/gregor.redelonghi/majstaf/majrcs/funclist-en

into

/home/gregor.redelonghi/majstaf/engit/majbin_en/git-commands/gupdate-file-en
/home/gregor.redelonghi/majstaf/engit/majbin_en/zze
/home/gregor.redelonghi/majstaf/engit/majrcs_en/funclist-en
